Class FuncRegressionEvaluator

Evaluator for the Function Regression task.

One continuous valued input maps to one continuous valued output.

Evaluation consists of querying the provided black box for a number of distinct values over the range of the continuous valued input, and comparing the black box response with the expected/correct response.
