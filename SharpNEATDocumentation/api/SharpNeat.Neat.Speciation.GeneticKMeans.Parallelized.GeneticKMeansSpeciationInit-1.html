Class GeneticKMeansSpeciationInit<T>

GeneticKMeansSpeciationStrategy initialisation.

This class handles initialising the k-means clusters (species), i.e. it takes a population of genomes and forms an initial set of k-means clusters with which to begin running the k-means iterations upon. upon.

Note. this implementation applies a modified version of the k-means++ initialisation method.
