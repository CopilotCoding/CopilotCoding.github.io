EnableHardwareAcceleratedActivationFunctions

Enable use of hardware accelerated neural network activation functions, i.e. alternate implementations that use CPU SIMD/vector instructions.

Declaration
public bool EnableHardwareAcceleratedActivationFunctions { get; set; }
Property Value
Type Description
bool
Remarks

The vectorized code is provided by alternative classes, and these classes tend to be more complex than their 'baseline' non-vectorized equivalents. Therefore when debugging a problem it is often useful to disable use of all vectorized code so as to rule out that code as the source of a problem/bug.

Furthermore, enabling hardware acceleration has been observed to often result in slower execution speed, probably because NEAT deals with non-homogeneous, irregular neural network structures that are generally not conducive to the application of vectorized code.
