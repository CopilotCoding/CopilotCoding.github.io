Creates new offspring from existing parent genomes.

Returns
Type Description
List<NeatGenome<T>>

A new list containing the created offspring genomes.
Remarks

Each species contains a genome list, which should have been trimmed back to the elite genomes only, in order to make space for the new offspring. New offspring are created by application of sexual and asexual reproduction on the existing genomes.

The number of offspring genomes to create is determined by OffspringAsexualCount and OffspringSexualCount.
