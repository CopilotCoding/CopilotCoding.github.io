Interface IPhenomeEvaluatorPool<TPhenome>

A pool of phenome evaluators.

Namespace: SharpNeat.Evaluation
Assembly: SharpNeat.dll
Syntax
public interface IPhenomeEvaluatorPool<TPhenome>
Type Parameters
Name Description
TPhenome

Phenome type.
Remarks

Used when the phenome evaluators have state, which is expensive to allocate, therefore we wish to maintain a pool or re-usable evaluators instead of constructing and discarding a new evaluator for each evaluation (or more accurately, each Parallel.For partition).
