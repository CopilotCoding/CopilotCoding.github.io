FindMedoid(IDistanceMetric<double>, ReadOnlySpan<ConnectionGenes<double>>)

int

Remarks

The coord with the lowest average distance from all other coords is the most central coord (the medoid). This method uses an inefficient N*N comparison of coords to find a medoid. It is provided only as a last resort for distance metrics for which no means exist to calculate a centroid.
