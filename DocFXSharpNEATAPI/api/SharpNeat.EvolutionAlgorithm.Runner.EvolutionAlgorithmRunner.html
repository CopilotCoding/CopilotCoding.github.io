Class EvolutionAlgorithmRunner

Wraps a background worker thread for running an IEvolutionAlgorithm, and provides methods for asynchronous control of the worker thread, for starting, stopping, pausing and resuming the evolution algorithm.

StartOrResume()

Starts the algorithm running, or resumes a paused algorithm.

Declaration
public void StartOrResume()
Remarks

This method can be called when the runner is in either the Ready or Paused states. If the runner is in the Paused state, then the runner is resumed. If the runner is already in the Running state, then the method does nothing and returns (and logs a warning). For all other states an exception is thrown.
